# 作业帮错题 PDF 处理

## 核心流程

```
输入 PDF  → 1. 读取全文识别知识点  → 2. 创建封面页  → 3. 正文页去品牌 + 加页眉页脚  → 输出
```

## 正文页处理

每页原文做两件事：

### 1. 删除品牌

```python
page.add_redact_annot(fitz.Rect(0, 0, PW, 68), fill=(1, 1, 1))
page.add_redact_annot(fitz.Rect(0, PH - 47, PW, PH), fill=(1, 1, 1))
page.apply_redactions()
```

- 顶部 y=0~68：覆盖 logo（~y=52）和分隔线，第一行题在 y=80
- 底部 y=795~842：覆盖页码（y=802），图片底边在 y=789 不被切

### 2. 叠加页眉提示页脚

全部居中绘制，在白色区域内：

| 元素 | y 位置 | 字号 | 颜色 |
|------|--------|------|------|
| 页眉 "日期 \| 知识点 \| 科目" | MP+14 | 9 | #595959 |
| 提示 "图形题用铅笔作图 \| 卡住时检查条件 \| 适当跳题" | MP+28 | 7 | #A6A6A6 |
| 页脚 "edsionc.top \| 邮箱" | PH-MP+4 | 8 | #8C8C8C |

## 知识点自动识别

扫描全文正则匹配，支持识别：

相似三角形、全等三角形、圆、切线、二次函数、一次函数、
正方形、矩形、梯形、平行四边形、菱形、等腰三角形、
直角三角形、等边三角形、三角函数、翻折对称、新定义

## 命名规范

输出文件名：`{学科}讲义-{yyyymmdd}.pdf`，同名自动加 `(1)` `(2)` 后缀。

## 用法

```bash
python3 process_mistake.py <输入PDF>
```

会自动从文件名解析学科和日期，无需手动指定。
